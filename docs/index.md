## Welcome to Quomponents

Include popular React components in your React projects easily with Quomponents.

## Installation (for standard modern project)

```markdown
npm i quomponents
```

##Usage
```markdown
import { AccentLine } from 'quomponents';

<AccentLine
asVariant="warning"
asSize="normal"
asFloated="inline"
withAnimation={{
animation: "zoom",
duration: 0.5,
delay: 0,
}}
isHidden={false}
isFluid={false}
/>
```
